06 Piaci Rések (White Spaces)

A. Middle Market Enterprise

Probléma: A piac polarizált — Palantir $1M+, Cursor $20/user. A $50k-250k szegmens üres.

Lehetőség: FDE-szerű bevezetés + platform, SMB/mid-market árazással. Custom skill/workflow építés, on-premise/hybrid.

B. FDE + Platform Kombináció

Probléma: Palantir FDE sikeres, de túl drága. Coding AI-knak nincs FDE. Frameworköknek nincs PS.

Lehetőség: Forward Deployed Engineering + capability platform. Helyszíni mérnökök 3-6 hónapra, knowledge transfer, ongoing support opció.

C. EU Regulációs Compliance

Probléma: AI Act 2026 augusztustól éles. Legtöbb US platform cloud-only. On-premise drága/hiányzik.

Lehetőség: On-premise/hybrid alapból, EU data residency, audit trail, MNB/DORA ready. Magyar/CEE lokális jelenlét.

D. Skill Ecosystem

Probléma: Coding AI-k zárt rendszerek. Frameworkök túl technikaiak. Enterprise platformok vendor lock-in.

Lehetőség: Nyílt skill/tool ecosystem (MCP). Low-code skill építés üzleti felhasználóknak. Partner ecosystem.

E. Legacy Rendszer Integráció

Probléma: Credal, Glean, Dust modern SaaS-hoz csatlakoznak (Slack, Google, Notion). Enterprise legacy rendszerek (Oracle, SAP, COBOL, custom fejlesztések) integrációját senki nem vállalja.

Lehetőség: Custom MCP server építés legacy rendszerekhez. FDE-k helyszínen elemzik és bekötik a rendszereket. Opcionálisan: legacy rendszerek AI-native újraírása managed service-ként.

08 Stratégiai Javaslatok

Rövid táv (0-6 hónap)

  • Pozicionálás: "AI Capability Partner for European Mid-Market" — nem chat wrapper, nem $1M+ platform
  • Go-to-market: 2-4 hetes "AI Bootcamp" (Palantir-inspired), $50-100k pilot szerződés
  • Technikai: MCP kompatibilitás, on-premise deployment, 3-5 vertical starter kit
  • Legacy Integráció: Custom MCP server építés ügyfelek legacy rendszereihez — ez a differenciáló

Középtáv (6-18 hónap)

  • Partner Ecosystem: System integrátor partnerségek (CEE), technológiai partnerségek (Windsurf/LangChain MCP)
  • Productization: Skill Marketplace, low-code skill builder, template library
  • Expansion: DACH piac, FSI vertical fokusz
  • Legacy Services: Opcionális legacy rendszer újraírás szolgáltatás — ahol az ügyfél ezt kéri, AI-native replacement

Hosszú táv (18+ hónap)

  • Platform Play: Ecosystem builder vs. point solution
  • Partner-led delivery: Recurring revenue focus
  • MCP ecosystem leadership a CEE régióban
